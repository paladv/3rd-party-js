# Weather
Chapter 1

## Install 

### node server app
Install node server app to serve third-party code

    npm install

## http server
Install simple http server to serve second-party static content

    npm install http-server -g

## Start

### node server app
Start node server app
    npm start

### http server
Start http-server on port 8080 using project directory as document root
    http-server ./ -p 8080

## Usage
Open [http://localhost:8080/](http://localhost:8080/) in browser
